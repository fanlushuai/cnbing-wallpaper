# cnbing-wallpaper
下载https://cn.bing.com/  每日壁纸..壁纸每天下载,且备份到本地,可以设置每天早上自动更换.
## 要求
 python3.5 , windows
## 使用
### 手动执行：
    点击run.bat可以直接执行。生成壁纸目录cnBingImage，以及备份目录
    
### 定时执行：
- 1、请严格将本程序放在D盘下。如：D:\cnBing\使用说明.txt
- 2、点击install，安装定时器
    
    注意：将windows桌面壁纸路径指向cnBingImage,设置windows壁纸自动切换,配合本程序task下载壁纸,
即可实现系统桌面背景每天切换
